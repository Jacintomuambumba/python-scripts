# Kanye Quotes Application!

- This project uses the TKinter Python module along with the requests module  to present random Kanye quotes found at "https://api.kanye.rest/" to the user using a GUI.

### Firstly clone the repo
```
git clone https://github.com/theadeyemiolayinka/python-scripts.git
```

### Then move into the directory
```
cd python-scripts/src/gh-adithya/Kanye Quotes GUI
```

### Install Requirements
```
pip install -r requirements.txt
```

### Now execute the program using python3
```
python3 main.py
```
